How It Works

Every vote is a head-to-head matchup. We use an Elo rating system — the same math used in chess — to build a community-powered ranking of all 50 candies. Beating a higher-ranked candy earns more points. All votes from all visitors feed into one shared leaderboard, so rankings improve over time.
