How It Works

Every vote is a head-to-head matchup between two Topps card designs. We use an Elo rating system — the same math used in chess — to build a community-powered ranking of all 73 designs (1952–2026). Beating a higher-ranked design earns more points. All votes from all visitors feed into one shared leaderboard, so rankings improve in accuracy over time. There are no accounts, no ads, just cards.
